# Netlify/React: Basic User Authentication Template

This template demonstrates **how to deploy** a React application that implements user authentication using Auth0 to Netlify. This React template builds the Single-Page Application (SPA) using the React Router 6 library.

This template is part of the ["Auth0 Developer Resources"](https://developer.auth0.com/resources), a place where you can explore the authentication and authorization features of the Auth0 Identity Platform.

Visit the ["How to Integrate a React App with Netlify and Auth0"](https://developer.auth0.com/resources/guides/cloud/netlify/basic-access-control) developer guide for instructions on how to quickly build and deploy secure React applications with Netlify and Auth0.

Click this button to deploy this React application secured with Auth0 to Netlify:

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/auth0-developer-hub/cloud_netlify_react_hello-world)

## Why Use Auth0?

Auth0 is a flexible drop-in solution to add authentication and authorization services to your applications. Your team and organization can avoid the cost, time, and risk that come with building your own solution to authenticate and authorize users. We offer tons of guidance and SDKs for you to get started and [integrate Auth0 into your stack easily](https://developer.auth0.com/resources/code-samples/full-stack).
